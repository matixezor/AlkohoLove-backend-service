# AlkohoLove-service

## Installing dependencies

Use `pip install -r requirements.txt`

## Running docker

Use `docker-compose up --build -d`  
Db `jdbc:postgresql://localhost:5432/alkoholove`

## Docs

`http://localhost:8008/docs`
